# dxvk-cache-tool

Standalone dxvk-cache merger

Usage
-----
```
dxvk-cache-tool [OPTION]... <FILEs>...

OPTIONS:    
        -o, --output FILE   Set output file name
        -h, --help          Display help and exit
        -V, --version       Output version information and exit
```

Building
-----
```
cargo build --release
target/release/dxvk-cache-tool
```
